UniConnect – University Event Management Platform

UniConnect is a full-stack university platform designed to simplify how students discover, register for, and manage campus events. The platform allows organizers to post events while students can browse opportunities across multiple domains and register easily.

This project is being developed as part of a database systems project and aims to provide a centralized platform for managing university events efficiently.

Project Overview

Universities host many events such as technical workshops, cultural programs, and non-technical competitions. However, students often miss opportunities due to scattered announcements across different platforms.

UniConnect solves this problem by creating a single platform where:

Organizers can post and manage events

Students can discover events easily

Registrations can be handled digitally

Event categories help students find relevant opportunities quickly

Features

Student Features

Student login using college email

Browse events across different categories

Register for events

View registered events in a personal dashboard

Organizer Features

Organizer login system

Create and publish events

Select event type (Technical, Non-Technical, Cultural)

Manage event details such as capacity and description

Platform Features

Event categorization

Simple and responsive interface

Database-driven event management

Scalable architecture for future improvements
